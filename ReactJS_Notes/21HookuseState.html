useState() | React

This hooks allows you to add state to functional components. it returns a state variable and a function to that variable.

We initialize our state by calling useState in our fucntion component.

useState accepts an initail state and returns two values:

  • The current state.
  • A function that updated the state.

What can state hold?

The useState hook can be used to keep track of

  • String
  • Numbers
  • Booleans
  • Arrays
  • Object
  • Any combination of these!

useState() | initialize

import React { useState } from "react";

function FavoriteColor (){
const [ // state variables , state updater function ] = useState("");

const [ color , setColor ] = useState("");
}

we are destructuring useState from react as it a named export.(useState returns two values)

Variables that can be named anything you would like

Read & update state

Read state

after initailize we can include our state anywhere in our component.

Update

To update our state, we use our state updater function.

Note:

We should never directly update state. Ex color = "red" is not allowed.

import React, { useState } from "react";

function HookUseState() {

// importing useState & initailizing
const [count, setCount] = useState(0);

// function for updating state by adding + 1
const increase = () => {
setCount (count + 1);
};

// function for updating state by subtracting - 1
const decrease = () => {
if (count < 0 ) {
return;
} else {
setCount (count - 1);
}
};

return (
<div>
<h1 style={{ color: lightblue }}>Counter | useState() </h1>
<h1>Counter : {count}</h1>
<button onclick={increase} >click to increase</button>
<button onclick={decrease} >click to decrease</button>
// calling state updater function
</div>
);
}

export default HookUseState;

import React, { useState } from "react";

function Bike() {
// use one State and include an Object
const [bike, setBike] = useState({ brand: "ducatti",
model: "panigale",
year: "2023",
color: "red"
});

// spreading the PreviousState and overwriting only the color
const updateColor = () => {
setBike(PreviousState =>{
return {...PreviousState, color: "black" }
});

return (
<>
<h1>Bikes</h1>
<p>
My fav bike is {bike.brand} {bike.model}
it is a {bike.color} from {bike.year}.
// read state from object
</p>
<button onclick={updateColor}>Black color</button>
// state update
</>
);
}

}

export default Bike;
