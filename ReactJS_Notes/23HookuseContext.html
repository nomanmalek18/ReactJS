useContext() | React

In React, the useContext hook is used to consume data from a React context. context provides a way to share data between components without having to pass props manually at every level of the component tree.

It allows you to create a global state that can be accessed b any component that needs it regardless of where it is in the component heirarchy.

How to use "useContext"

1. Create the context

First, you need to create context using the React,createContext()method. this method returns an object with tow properties: provider and consumer.

Wrap the component(s) that should have access to the shared data with the provider, this is typically done at the highest level in component tree.
