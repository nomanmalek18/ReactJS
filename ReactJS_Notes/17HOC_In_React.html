HOC (Higher Order Component)

Whats Is HOC?

In React, a Higher-Order Component (HOC) is a function that takes a component as an input and returns a new enhanced component as output.

It's like a special function that can make your components more powerful and add extra functionality to them.

Without HOC

import React, { useState } from "react"; function Item2() { const [cost, setCost] = useState(10); let costHandler() =>{ setCost(cost. 10); }; return ( <div className="item2"> <h1> Item2 Cost: <span>*{cost}</span> </h1> <button onClick={costHandler}> Increase Cost </button> </div> ); } export default Item2;

HOC Function

1 2 3 4 5 6
import React, { useState } from "react"; function UpdatedComponent(OriginalComponent) { function NewComponent() { const [cost, setCost] = useState(10); let costHandler = () =>{ setCost(cost + 10); }; return ( <OriginalComponent costHandler={costHandler} cost={cost} /> ) } return NewComponent; } export default UpdatedComponent;
