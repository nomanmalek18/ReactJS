Introduction of ReactJS

What Is ReactJs

ReactJS is an open-source JavaScript library used to create user interfaces in a declarative and efficient way. It is a component-based front-end library responsible only for the view layer of a Model View Controller(MVC) architecture. React is used to create modular user interfaces and promotes the development of reusable UI components that display dynamic data. it was created by Jordan Walke, a Software Engineer at the Facebook, the person affected by XHP

  • React is a JavaScript library created by Facebook
  • React is a User Interface (UI) library
  • React is a tool for building UI component

the React was First deployedfor Facebook's Newsfeed application in 2011, and then deployed for instagram.com in 2012.

it was open-sourced at JSConf US in May 2013

back in 2011, the develoeprs at Facebook started to face some issues with code maintenance. as the Facebook ads app got an increasing number of feature, the team needed more people to keep it running flawlessly.

The growing number of team-members and app-features slowed them down as a company. over time, their app became difficult to handleas they faced a lot of cascading updates.

after a while, Engineer at Facebook couldn't keep up with these cascading updates. their code demanded an urgent upgrade to become more efficient.

they had the Model right, but they needed to do something about user experience. so, jorden walke built a prototype that made the process more efficient, and this marks the birth of React.Js.
